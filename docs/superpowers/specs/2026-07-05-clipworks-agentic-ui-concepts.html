方案对比矩阵

绿色高亮为当前推荐方案。点击卡片可查看该方向的示意草图。

1. 交互模型

Chat-first
所有操作通过对话完成，界面极简，但对复杂编辑不够直观。
Agentic Studio
每个 UI 区域都有 AI 入口，适合专业用户，但学习成本略高。
Wizard
一步一步引导，最小白友好，但灵活性不足，高级用户会觉得慢。
推荐
Hybrid 混合式
对话为主，节点精准控制，兼顾简单与灵活。

2. 修改锚点

推荐
场景卡片
按画面拆分，用户看到什么就能改什么，最符合直觉。
时间线点选
专业剪辑习惯，但对小白有门槛。
纯对话
无需 UI 元素，但用户需要自己描述位置，容易出错。
预览区圈选
最有未来感，但实现复杂，MVP 不建议。

3. 首页入口

推荐
Agent Launchpad
一句话开始创作，最符合「AI 视频工厂」心智。
项目列表 + AI 创建
传统模式，适合管理历史项目，但缺乏冲击力。
双模式切换
兼顾两种用户，但增加认知负担。
对话式浏览器
用自然语言管理项目，新颖但不够直接。

4. 工作区布局

推荐
三栏影院式
左 Agent/场景，中预览，右属性，信息结构最清晰。
对话流式
像 Claude Artifacts，聊天即工作流，但预览尺寸受限。
全屏预览 + 底部条
沉浸感最强，但 Agent 和场景信息被压缩。
分阶段动态布局
叙事感强，但实现和动画成本高。

5. 生成反馈

步骤卡片
清晰但不吵闹，适合大多数场景。
推荐
可视化流水线
契合「映工厂」品牌，视觉冲击强，用户能感知 AI 在做什么。
Agent 日志流
最有智能体感，但信息噪音可能较大。
极简进度条
干净，但缺乏 Agentic 感。
